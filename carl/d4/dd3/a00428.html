Detailed Description

Represent a dynamic root, also known as a "root expression". It's a generalization of an algebraic real(= univariate poly having roots + root index). It uses a multivariate poly with distinguished, root-variable "_z" and an root index. If we want to "evaluate" this root, we need to plug in a subpoint with algebraic-real values for all variables that are not "_z". The result is basically an algebraic real. And since this algebraic real depends on the subpoint, you can also think of this as a multidimensional scalar function: A root(p(x1,x2,_z), 1) value represents a function f(x1,x2) of algReal^2 to an algebraic real. For example, f(a,b)= root(q(_z),1), where the resulting algebraic real is the 1. root of poly q(_z) := p(a,b,_z) after pluggin a and b into poly p.
