141  /**
142  * Implements a union-find datastructure, a datastructure containing an array of entries.
143  * Basically, this is an array of elements of type T with the following extra structure:
144  * Every entry is contained in an equivalence class identified by a single representative entry.
145  * In order to find this representative, every element has an associated parent entry;
146  * if this is the entry itself, the entry is the representative of its equivalence class.
147  * Using a rank count (basically a lower bound for the height of the tree below an entry),
148  * the height of the trees below the representative entries is ALWAYS less than ceil(log2(#entries)).
149  * Moreover, while finding the parent of an entry, all entries on the path to the root are set
150  * to directly point to the root element. This improves the amortized number of steps used to find
151  * the representative of an entry from O(log2(n)) to O(a(n)) where a(n) is the inverse of the Ackermann function.
152  * All this extra structure is embedded directly within the array.
153  */

155  typename T = void, // the type to carry around with us
156  bool no_compression = true, // turn on/off compression of rank and class index into the same 8-byte variable, defaults to off (seems to be faster)
