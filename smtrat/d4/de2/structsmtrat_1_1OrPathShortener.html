Detailed Description

A rewriter that checks every (X)OR that only contains ANDs and UEQs as children for equalities that hold for every alternative, and adds such equalities explicitly.

This should run after NNF transformation (but clearly before CNF transformation). This defeats eq_diamond examples (and also more general cases where the paths are longer or irregular).
